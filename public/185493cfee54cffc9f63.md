# Next.jsとClaude APIで作れる便利ツール3選

プログラミング初心者の皆さん、Next.jsとClaude APIを使って実用的なツールを作ってみませんか？今回は、比較的簡単に作れて実際に役立つ3つのツールをご紹介します。

## 1. 文章要約ツール

長い文章を短時間で理解したいとき、Claude APIの力を借りて要約ツールを作ることができます。

### 基本的な実装例

```typescript
// pages/api/summarize.ts
import { NextApiRequest, NextApiResponse } from 'next';

export default async function handler(
  req: NextApiRequest,
  res: NextApiResponse
) {
  if (req.method !== 'POST') {
    return res.status(405).json({ error: 'Method not allowed' });
  }

  const { text } = req.body;

  try {
    const response = await fetch('https://api.anthropic.com/v1/messages', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': process.env.CLAUDE_API_KEY!,
        'anthropic-version': '2023-06-01'
      },
      body: JSON.stringify({
        model: 'claude-3-haiku-20240307',
        max_tokens: 300,
        messages: [{
          role: 'user',
          content: `以下の文章を3行程度で要約してください：\n\n${text}`
        }]
      })
    });

    const data = await response.json();
    const summary = data.content[0].text;

    res.status(200).json({ summary });
  } catch (error) {
    res.status(500).json({ error: '要約処理でエラーが発生しました' });
  }
}
```

このツールは、ニュース記事や論文、長いメールなどを素早く理解したいときに重宝します。

## 2. 英語学習チャットボット

英語の練習相手として、Claude APIを活用したチャットボットを作成できます。文法の間違いを指摘してくれたり、自然な表現を教えてくれる優秀な先生になります。

### 特徴
- リアルタイムでの英会話練習
- 文法ミスの自動訂正
- シチュエーション別の会話練習（レストラン、空港、ビジネスなど）
- 発音のアドバイス（テキストベース）

実装では、Claudeに「英語教師として振る舞い、間違いを優しく訂正する」といったシステムプロンプトを設定することで、学習に最適な応答を得られます。
