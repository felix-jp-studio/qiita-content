# 初心者がつまずくTypeScriptのエラー5選と解決策

TypeScriptを始めたばかりの頃は、見慣れないエラーメッセージに悩まされることが多いと思います。今回は、初心者がよく遭遇するTypeScriptのエラー5つと、その解決策をわかりやすく解説します。

## 1. Type 'string | undefined' is not assignable to type 'string'

このエラーは、値が `undefined` の可能性があるときに発生します。

```typescript
// ❌ エラーが発生するコード
const userName: string = localStorage.getItem('username'); // Error!

// ✅ 解決策1: 非null演算子を使用
const userName: string = localStorage.getItem('username')!;

// ✅ 解決策2: デフォルト値を設定
const userName: string = localStorage.getItem('username') || 'Guest';

// ✅ 解決策3: 条件分岐で処理
const userNameOrNull = localStorage.getItem('username');
if (userNameOrNull) {
  const userName: string = userNameOrNull;
  console.log(userName);
}
```

## 2. Property does not exist on type

オブジェクトに存在しないプロパティにアクセスしようとしたときのエラーです。

```typescript
// ❌ エラーが発生するコード
const user = { name: 'John', age: 25 };
console.log(user.email); // Error: Property 'email' does not exist

// ✅ 解決策: 型定義を明確にする
interface User {
  name: string;
  age: number;
  email?: string; // オプショナルプロパティ
}

const user: User = { name: 'John', age: 25 };
console.log(user.email); // undefined（エラーなし）
```

## 3. Argument of type is not assignable to parameter

関数の引数の型が合わないときのエラーです。

```typescript
// ❌ エラーが発生するコード
function greet(name: string) {
  console.log(`Hello, ${name}!`);
}

const userId = 123;
greet(userId); // Error!

// ✅ 解決策: 型を変換する
greet(userId.toString());

// または関数の型定義を変更
function greetFlexible(name: string | number) {
  console.log(`Hello, ${name}!`);
}
```

## 5. Object is possibly 'null' or 'undefined'

DOM要素を取得するときによく発生するエラーです。

```typescript
// ❌ エラーが発生するコード
const button = document.getElementById('submit-btn');
button.addEventListener('click', () => {}); // Error!

// ✅ 解決策1: 非null演算子
const button = document.getElementById('submit-btn')!;
button.addEventListener('click', () => {});

// ✅ 解決策2: 条件分岐（推奨）
const button = document.getElementById('submit-btn');
if (button) {
  button.addEventListener('click', () => {
    console.log('Button clicked!');
  });
}

// ✅ 解決策3: オプショナルチェイニング
const button = document.getElementById('submit-btn');
button?.addEventListener('click', () => {
  console.log('Button clicked!');
});
```
