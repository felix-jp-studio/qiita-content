# AWS初心者がまず覚えるべきサービス5選

AWSを始めたいけれど、200以上あるサービスの中からどれを学べばいいか迷っていませんか？この記事では、AWS初心者が最初に覚えるべき5つのコアサービスを厳選してご紹介します。

## 1. EC2（Elastic Compute Cloud）

**仮想サーバーサービス**

EC2はAWSの心臓部とも言える仮想サーバーサービスです。Webサイトやアプリケーションを動かすためのコンピューターを、クラウド上で簡単に作成・管理できます。

- 用途：Webサーバー、アプリケーションサーバー
- 特徴：必要な時に必要な分だけ使える従量課金制

## 2. S3（Simple Storage Service）

**オブジェクトストレージサービス**

S3は画像、動画、ドキュメントなど、あらゆるファイルを保存できるストレージサービスです。99.999999999%（イレブンナイン）の耐久性を誇り、データを安全に保管できます。

```bash
# AWS CLIを使用したS3へのファイルアップロード例
aws s3 cp myfile.txt s3://my-bucket/
aws s3 ls s3://my-bucket/
```

- 用途：静的ファイルの保存、バックアップ、静的サイトホスティング
- 特徴：容量無制限、高い可用性

## 3. RDS（Relational Database Service）

**マネージドデータベースサービス**

RDSは、MySQL、PostgreSQL、Oracleなどの関係データベースを簡単に構築・運用できるサービスです。データベースの管理作業（バックアップ、パッチ適用など）を自動化してくれます。

- 用途：アプリケーションのデータ保存
- 特徴：自動バックアップ、マルチAZ構成による高可用性

## 4. Lambda

**サーバーレスコンピューティングサービス**

Lambdaは、サーバーを管理することなくコードを実行できるサービスです。必要な時だけ動作し、使った分だけ課金される非常にコスト効率の良いサービスです。

```python
# Lambda関数の例（Python）
import json

def lambda_handler(event, context):
    name = event.get('name', 'World')
    message = f'Hello, {name}!'
    
    return {
        'statusCode': 200,
        'body': json.dumps({
            'message': message
        })
    }
```

- 用途：API処理、画像リサイズ、定期的なタスク実行
- 特徴：サーバー管理不要、自動スケーリング

## 5. CloudFormation

**インフラストラクチャ・アズ・コード（IaC）サービス**

CloudFormationは、AWSのインフラをコードで管理できるサービスです。JSONやYAMLファイルでインフラを定義し、一括でリソースを作成・管理できます。

- 用途：インフラの自動構築、環境の複製
- 特徴：バージョン管理可能、ロールバック機能
