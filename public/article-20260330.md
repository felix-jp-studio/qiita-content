# Next.jsとClaude APIで作れる便利ツール3選

プログラミング初心者の皆さん、Next.jsとClaude APIを組み合わせて便利なツールを作ってみませんか？今回は、比較的簡単に作れて実用性の高い3つのツールをご紹介します。

## 1. 文章要約ツール

長い文章を短時間で理解したいときに便利な要約ツールです。学習資料やニュース記事の要点を素早く把握できます。

### 基本的な実装例

```typescript
import { useState } from 'react';

export default function SummaryTool() {
  const [inputText, setInputText] = useState('');
  const [summary, setSummary] = useState('');
  const [loading, setLoading] = useState(false);

  const handleSummarize = async () => {
    setLoading(true);
    try {
      const response = await fetch('/api/summarize', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ text: inputText }),
      });
      const data = await response.json();
      setSummary(data.summary);
    } catch (error) {
      console.error('要約に失敗しました:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="max-w-2xl mx-auto p-6">
      <h1 className="text-2xl font-bold mb-4">文章要約ツール</h1>
      <textarea
        value={inputText}
        onChange={(e) => setInputText(e.target.value)}
        placeholder="要約したい文章を入力してください"
        className="w-full h-40 p-3 border rounded-lg mb-4"
      />
      <button
        onClick={handleSummarize}
        disabled={loading || !inputText}
        className="bg-blue-500 text-white px-6 py-2 rounded-lg hover:bg-blue-600 disabled:opacity-50"
      >
        {loading ? '要約中...' : '要約する'}
      </button>
      {summary && (
        <div className="mt-6 p-4 bg-gray-100 rounded-lg">
          <h3 className="font-semibold mb-2">要約結果:</h3>
          <p>{summary}</p>
        </div>
      )}
    </div>
  );
}
```

## 2. 英語学習チャットボット

英会話の練習ができるチャットボットです。Claude AIが自然な会話相手となり���英語力向上をサポートします。

### 特徴
- **レベル別対応**: 初級から上級まで、ユーザーのレベルに合わせた会話
- **文法修正**: 間違いがあれば優しく修正してくれる
- **トピック選択**: 日常会話からビジネス英語まで幅広いトピックに対応

実装では、Claude APIに「英語学習のサポートをする」というロールを与え、ユーザーの英語レベルに応じた返答をするよう指示します。

## 3. コードレビューアシスタント

プログラミング学習者にとって心強い味方となるコードレビューツールです。書いたコードの改善点や最適化のアドバイスを受けられます。

### 機能
- **バグの検出**: 潜在的な問題を指摘
- **パフォーマンス改善**: より効率的なコードの提案
- **ベストプラクティス**: 業界標準に沿った書き方のアドバイス
- **セキュリティチェック**: セキュリティホールの指摘

ユーザーがコードを貼り付けると、Claude APIが詳細な分析を行い、具体的な改善提案を返してくれます。

## 開発のポイント

これらのツールを開発する際は、以下の点に注意しましょう：

1. **エラーハンドリング**: API呼び出しが失敗した場合の処理
2. **レスポンシブデザイン**: モバイルでも使いやすいUI
3. **ローディング状態**: ユーザーに処理中であることを伝える
4. **入力検証**: 不正な入力を防ぐバリデーション
