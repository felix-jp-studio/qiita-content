# Next.jsとClaude APIで作れる便利ツール3選

プログラミング初心者の皆さん、こんにちは！今回は、Next.jsとClaude APIを組み合わせて作れる実用的なツールを3つご紹介します。これらのツールは、日常業務やプログラミング学習に役立つこと間違いなしです。

## 1. 文章校正・リライトツール

ブログ記事やメール文章の校正・リライトを自動化できるツールです。文章を入力するだけで、より読みやすく魅力的な文章に変換してくれます。

**主な機能**
- 誤字脱字チェック
- 文章の自然性向上
- トーン調整（丁寧語、カジュアルなど）

```typescript
// pages/api/rewrite.ts
import { NextApiRequest, NextApiResponse } from 'next';
import Anthropic from '@anthropic-ai/sdk';

const anthropic = new Anthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

export default async function handler(
  req: NextApiRequest,
  res: NextApiResponse
) {
  const { text, tone } = req.body;

  try {
    const message = await anthropic.messages.create({
      model: "claude-3-haiku-20240307",
      max_tokens: 1000,
      messages: [{
        role: "user",
        content: `以下の文章を${tone}なトーンでリライトしてください：\n${text}`
      }]
    });

    res.status(200).json({ rewrittenText: message.content });
  } catch (error) {
    res.status(500).json({ error: 'リライトに失敗しました' });
  }
}
```

## 実装のポイント

これらのツールを作る際の重要なポイントをご紹介します：

### 1. 環境変数の設定
Claude APIキーは必ず環境変数で管理しましょう。`.env.local`ファイルに以下のように記載します：

```
ANTHROPIC_API_KEY=your_api_key_here
```

### 2. エラーハンドリング
APIコールは必ず失敗する可能性があるため、適切なエラーハンドリングを実装することが重要です。

### 3. ユーザビリティの向上
- ローディング表示
