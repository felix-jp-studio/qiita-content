TypeScriptを始めたばかりの初心者の方は、JavaScriptでは見たことのないエラーメッセージに戸惑うことが多いでしょう。この記事では、初心者がよく遭遇するTypeScriptのエラー5つと、その解決策を分かりやすく解説します。

## 1. Property does not exist on type

このエラーは、オブジェクトに存在しないプロパティにアクセスしようとした時に発生します。

```typescript
// エラーが発生するコード
const user = { name: "太郎" };
console.log(user.age); // Property 'age' does not exist on type '{ name: string; }'

// 解決策1: プロパティを追加
const user = { name: "太郎", age: 25 };
console.log(user.age);

// 解決策2: オプショナルアクセス
console.log(user.age || "年齢不明");
```

**解決策**：
- 必要なプロパティをオブジェクトに追加する
- インターフェースを定義して���を明確にする

## 2. Type 'string' is not assignable to type 'number'

型の不一致で発生する最も基本的なエラーです。

```typescript
// エラーが発生するコード
let age: number = "25"; // Type 'string' is not assignable to type 'number'

// 解決策
let age: number = 25; // 数値型で代入
// または
let age: number = Number("25"); // 文字列を数値に変換
```

**解決策**：
- 変数の型定義を正しく行う
- 必要に応じて型変換を行う

## 3. Object is possibly 'null'

nullの可能性があるオブジェクトにアクセスしようとした時のエラーです。

```typescript
// エラーが発生するコード
const element = document.getElementById("button");
element.addEventListener("click", () => {}); // Object is possibly 'null'

// 解決策1: null チェック
if (element) {
  element.addEventListener("click", () => {});
}

// 解決策2: オプショナルチェイニング
element?.addEventListener("click", () => {});
```

**解決策**：
- null チェックを行う
- オプショナルチェイニング（`?.`）を使用する

## 4. Argument of type is not assignable to parameter

関数の引数に間違った型を渡した時のエラーです。

```typescript
// エラーが発生するコード
function greet(name: string) {
  console.log(`Hello, ${name}`);
}

greet(123); // Argument of type 'number' is not assignable to parameter of type 'string'

// 解決策
greet("太郎"); // 正しい型で引数を渡す
// または
greet(String(123)); // 型変換を行う
```

**解決策**：
- 関数の型定義を確認し、正しい型の引数を渡す
- 必要に応じて型変換を行う

## 5. Cannot find module

モジュールのインポート時に発生するエラーです。

```typescript
// エラーが発生するコード
import { myFunction } from './nonexistent'; // Cannot find module './nonexistent'

// 解決策1: 正しいパスを指定
import { myFunction } from './utils'; // 存在するファイルをインポート

// 解決策2: 型定義ファイルの作成（外部ライブラリの場合）
// npm install @types/library-name
```

**解決策**：
- ファイルパスが正しいか確認する
- 外部ライブラリの場合は型定義ファイルをインストールする
