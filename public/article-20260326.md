# Firebase×Next.jsで認証機能を30分で実装する手順

Webアプリケーションに認証機能を実装するのは、初心者にとって難しそうに感じるかもしれません。しかし、FirebaseとNext.jsを組み合わせることで、驚くほど簡単に実装できます。

この記事では、メール・パスワード認証を30分で実装する手順を解説します。

## Firebaseとは？

Firebaseは、Googleが提供するアプリケーション開発プラットフォームです。認証、データベース、ホスティングなどの機能を簡単に利用できます。

## 必要な準備

- Node.js（v16以上）
- Firebaseプロジェクトの作成
- Next.jsプロジェクトの準備

## Step 1: プロジェクト設定

まず、Firebase SDKをインストールします。

```bash
npm install firebase
```

## Step 2: Firebase設定ファイル作成

��ロジェクトルートに`lib/firebase.js`を作成し、Firebase設定を記述します。

```javascript
import { initializeApp } from 'firebase/app';
import { getAuth } from 'firebase/auth';

const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-project-id.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project-id.appspot.com",
  messagingSenderId: "123456789",
  appId: "your-app-id"
};

const app = initializeApp(firebaseConfig);
export const auth = getAuth(app);
```

## Step 3: 認証コンポーネント作成

`components/Auth.js`でログイン・サインアップフォームを作成します。

```javascript
import { useState } from 'react';
import { signInWithEmailAndPassword, createUserWithEmailAndPassword } from 'firebase/auth';
import { auth } from '../lib/firebase';

export default function Auth() {
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');
  const [isLogin, setIsLogin] = useState(true);

  const handleSubmit = async (e) => {
    e.preventDefault();
    try {
      if (isLogin) {
        await signInWithEmailAndPassword(auth, email, password);
      } else {
        await createUserWithEmailAndPassword(auth, email, password);
      }
      alert('認証成功！');
    } catch (error) {
      alert('エラー: ' + error.message);
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <h2>{isLogin ? 'ログイン' : 'サインアップ'}</h2>
      <input
        type="email"
        placeholder="メールアドレス"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        required
      />
      <input
        type="password"
        placeholder="パスワード"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
        required
      />
      <button type="submit">
        {isLogin ? 'ログイン' : 'サインアップ'}
      </button>
      <button type="button" onClick={() => setIsLogin(!isLogin)}>
        {isLogin ? 'アカウント作成' : 'ログインに戻る'}
      </button>
    </form>
  );
}
```

## Step 4: 認証状態の管理

`pages/_app.js`で認証状態を管理します。

```javascript
import { useEffect, useState } from 'react';
import { onAuthStateChanged, signOut } from 'firebase/auth';
import { auth } from '../lib/firebase';
import Auth from '../components/Auth';

export default function App({ Component, pageProps }) {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    const unsubscribe = onAuthStateChanged(auth, (user) => {
      setUser(user);
      setLoading(false);
    });

    return () => unsubscribe();
  }, []);

  if (loading) return <div>読み込み中...</div>;

  if (!user) {
    return <Auth />;
  }

  return (
    <div>
      <p>ようこそ、{user.email}さん！</p>
      <button onClick={() => signOut(auth)}>ログアウト</button>
      <Component {...pageProps} />
    </div>
  );
}
```

## まとめ

これで基本的な認証機能の実装は完了です！メール・パスワード認証、ログアウト機能、認証状態の管理が動作します。

Firebase認証の優れた点は、セキュリティやパスワードの暗号化を自動で処理してくれることです。初心者でも安全な認証システムを簡単に構築できます。
